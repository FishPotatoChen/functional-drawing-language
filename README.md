# 函数绘图语言-编译原理

## 使用说明

运行`main.py`文件，对应的文件夹中文件就是相对应的意思，`test.txt`是输入，其中的`ppt`就是解释。`output.py`是编译结果。

既可以用当做编译器，又可以当做解释器。
